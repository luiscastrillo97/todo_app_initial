## Instalación

Clonar el repositorio
`git clone https://github.com/luiscastrillo97/todo_app_initial.git`

Moverse al directorio del proyecto
`cd todo_app_initial`

Instalar dependencias
`npm install`

Ejecutar el proyecto
`npm run dev`
